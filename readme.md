# Deno: Crear una API Rest usando JWT

## Instalación

- Primero instala [denon](https://deno.land/x/denon/)

## Ejecutar

Luego de instalar `denon`, ejecuta el siguiente comando:

`denon start`

## Actualizar Cache

`deno run -r app.ts` y luego `denon start`
